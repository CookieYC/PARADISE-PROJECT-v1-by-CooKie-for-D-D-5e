# Paradise Project — Неофициальный сеттинг для D&D 5e

**«Парадиз»** — это постапокалиптический сеттинг-конструктор для Dungeons & Dragons 5-й редакции.  
Нефть кончилась. Кристаллы икора дают энергию, но порождают чудовищ. Ляписы — абсолютное зло с человеческим лицом — скрываются среди людей. А боги, возможно, не умерли. Возможно, они просто спят.

## Что это?

Это **инструментарий для Мастера (DM)**, а не готовый мир с картами и королями.  
Вы получаете:

- 5 уникальных классов (Инженер, Костолом, Гладиатор, Охотник, Артист)
- Систему Божественных Даров (250+ способностей)
- Полный бестиарий существ икора (эфиры, мутанты, конструкты, ляписы, ангелы, серафимы)
- Правила выживания и заражения икором
- Инструменты для быстрой генерации встреч, лута и NPC
- Каталог оружия (включая огнестрел) и экономику на патронах

Всё это свёрстано в единый HTML-файл (`PARADISE_PROJECT_by_CooKie.html`) в стиле, близком к dnd.su.

## Для кого?

- **Для опытных мастеров**, которые ищут мрачный, нестандартный сеттинг с глубокими механиками.
- **Для новичков**, которые хотят начать играть без тонны официальных книг.

## Как использовать?

1. Скачайте этот репозиторий (Code → Download ZIP).
2. Распакуйте архив в любую папку.
3. Откройте файл `PARADISE_PROJECT_by_CooKie.html` в браузере — это вся библия сеттинга.
4. Изучайте, меняйте правила под себя, создавайте свои города, фракции и кампании.
5. Играйте!

## Жанры

Постапокалипсис, техно-фэнтези, стимпанк, выживание, хоррор, психологическая драма, приключение.
